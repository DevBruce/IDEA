사용방법

1. 자동생성모드에 해당하므로 생성시간이 최소 10초 소모됩니다.

2. Max Word Size 는 워드클라우드에서 가장 큰(빈도수가 가장높은) 단어의 크기입니다.
사용자가 원하는 크기를 입력하시면 됩니다.

3. Bg Color 는 워드클라우드 배경의 색상을 설정하는 값입니다.
사용자가 희망하는 색상을 선택하시면 됩니다.

4. Shape 는 워드클라우드의 모양을 의미합니다.
사용자가 희망하는 모양을 선택하시면 됩니다.

5. Page Range 는 웹에서 데이터를 크롤링할 페이지의 갯수를 의미합니다.
기본값은 3이며, 1~3 페이지의 데이터를 크롤링 한다는 것을 의미합니다.
(1 증가당 약 3초 증가)

6. Stop Words (불용어) 는 결과값에서 제외할 단어를 의미합니다.
결과값에서 제외할 단어가 복수일 경우 ,(콤마)로 구분해주시길 바랍니다.
