사용방법

1. 자동생성모드에 해당하므로 생성시간이 최소 10초 소모됩니다.

2. Edge Remove Threshold 는 default 값이 0입니다.
이 경우 각 단어사이의 edge가 제거없이 모두 표시됩니다.
하지만 글의 수가 많아질 경우 edge 의 수를 줄여주어야 관계를 파악하는데 수월합니다.
포스트의 갯수가 100개 미만일 경우 0, 100개이상 2, 1000개이상 4 정도가 적당합니다.
지나치게 큰 값은 오류를 일으키므로 적절하게 조절해주세요.

3. Node Num 은 화면에 표시될 노드의 갯수입니다.
본인이 원하는 갯수를 입력하시면 됩니다.
하지만 지나치게 큰 수를 입력하게 된다면 결과값이 왜곡될 수 있으니 주의해주세요.

4. Page Range 는 웹에서 데이터를 크롤링할 페이지의 갯수를 의미합니다.
기본값은 3이며, 1~3 페이지의 데이터를 크롤링 한다는 것을 의미합니다.
(1 증가당 약 3초 증가)

5. Stop Words (불용어) 는 결과값에서 제외할 단어를 의미합니다.
결과값에서 제외할 단어가 복수일 경우 ,(콤마)로 구분해주시길 바랍니다.
